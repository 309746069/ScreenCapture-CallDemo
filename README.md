# NiuniuCaptureCallDemo
一个高品质的Windows桌面截图控件，ScreenCapture, Demo source code to show how to call NiuniuCapture.dll

牛牛截图现已完美支持多显示器及DPI放大的场景，欢迎试用。
 
在上一个版本发布后，收到非常多朋友的反馈，近期有了些时间，对控件进行了升级；同时对比测试了几大主流的截图功能，除了腾讯QQ最新版，其他如360、旺旺、人生日历截图等，均对DPI放大的支持不是很好，多显示器支持我仅测试了腾讯，其他的实现不清楚。

完整的介绍可以访问：http://blog.csdn.net/tragicguy/article/details/43998115

以下是牛牛截图控件2.0的相关介绍： <br/>
1. 本控件实现的基本功能及特点如下<br/>
1).支持窗口区域的自动识别<br/>
2).支持矩形、圆形、箭头、文字、画刷、荧光笔的绘制以及撤销<br/>
3).可另存为png、jpg、bmp等格式<br/>
4).支持鼠标所在区域的图像放大功能，以便精准定位<br/>
5).窗口大小、鼠标坐标显示以及光标处的颜色拾取<br/>
6).采用C++语言开发的Win32动态库，大小为152KB<br/>
7).支持XP、WIN7、WIN8及WIN2003、WIN2008等操作系统<br/>
8).此控件提供标准的接口，方便集成进其他软件中<br/>
例如可以与浏览器控件整合，实现Web页面截图的功能，也可以与其他如即时通讯等其他任何系统中<br/>
 
2. 本次控件升级的更新项<br/>
1). 修复了原有文字功能bug，防止文字丢失的情况<br/>
2). 修复了对设置了透明属性的窗口不能截取的bug<br/>
3). 增加了荧光笔功能<br/>
4). 优化了箭头功能<br/>
5). 扩展了放大镜的显示效果，一种是跟随鼠标，一种是跟随选中区域<br/>
6). 其他一此细节体验优化<br/>
7). 增加对保存时的默认文件名进行设置的功能<br/>
8). 增加对多显示器的支持<br/>
9). 增加对DPI放大的支持 [ 需要在调用程序中设置 ]<br/>
